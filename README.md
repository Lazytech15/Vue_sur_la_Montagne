# Vue sur la Montagne Resort & Spa — Hotel Landing Page

A luxury hotel management landing page for a resort in Rizal, Philippines.
Built with **React + Vite + Tailwind CSS**.

---

## ✨ Features

- **Stunning animated hero** with SVG mountain/forest landscape of Rizal
- **Live booking form** with check-in/out dates, room type, guests selection
- **6 room types** with illustrated SVG panels and hover interactions
- **8 curated services** showcasing Rizal-specific experiences
- **SVG Illustrated gallery** with 6 Rizal landscapes
- **Interactive map** of Rizal province with location markers
- **Testimonials carousel** with smooth transitions
- **Contact form** with validation
- **Fully responsive** for mobile, tablet, and desktop
- **Google Fonts** — Cormorant Garamond + DM Sans + Cinzel

---

## 🚀 Getting Started

### Prerequisites
- Node.js 18+ required

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
```
